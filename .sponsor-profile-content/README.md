# Sponsor Profile Content

Working copies of the text used on the GitHub Sponsors profile
(https://github.com/sponsors/JPF1111).

- `SHORT_BIO.txt` — the 250-char short bio
- `INTRODUCTION.md` — the full Sponsors profile introduction

These are kept in version control so the canonical sponsor copy lives
alongside the project. Edits here do not automatically sync to the
Sponsors profile — update both when the messaging changes.

Excluded from the published GitHub Pages site via `_config.yml`.
