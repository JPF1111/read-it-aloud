# Chrome Web Store Assets

Source-of-truth marketing assets for the Read It Aloud Chrome Web Store listing.
All screenshots are captured from the live extension (v1.3.4) running on a real
Wikipedia article; promo graphics use the project's brand system (dark canvas,
amber to pink to purple waveform gradient, Big Shoulders wordmark).

## Screenshots (1280x800, RGB, no alpha)

| File | Shows |
|------|-------|
| `01_hero_dark_player.png` | Dark mini-player mid-playback with follow-along text and a highlighted selection — the hero shot |
| `02_popup_voices.png` | Toolbar popup (Text to Audio tab) alongside the mini-player |
| `03_popup_settings.png` | Toolbar popup Settings tab (speed, stability, similarity, style sliders) |
| `04_floating_button.png` | The floating "Read aloud" button appearing on a text selection |

Upload order in the store is by filename (01 first = most prominent).

## Promo graphics

| File | Size | Use |

| `ria_promo_tile_440x280.png` | 440x280 | Small promo tile (required for featuring) |
