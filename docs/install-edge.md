# Install Read It Aloud on Microsoft Edge & Opera

Both Edge and Opera are Chromium-based browsers that accept Chrome
extensions, so the Chrome version of Read It Aloud works on both.

---

## Microsoft Edge

### Option 1: Microsoft Edge Add-ons store *(preferred when live)*

> 🔜 **Status:** A dedicated Edge Add-ons listing is in progress. When
> published, it will be the recommended install path for Edge users.
> Star the [GitHub repository](https://github.com/JPF1111/read-it-aloud)
> to be notified.

### Option 2: Install from the Chrome Web Store

Edge supports Chrome Web Store extensions out of the box:

1. Open Microsoft Edge
2. Go to the [Chrome Web Store listing](https://github.com/JPF1111/read-it-aloud) *(direct link coming soon)*
3. Edge will prompt you to allow extensions from other stores — click **Allow**
4. Click **Get** to install
5. Confirm the permissions dialog

---

## Opera

Opera also supports Chrome Web Store extensions:

1. Install the [Install Chrome Extensions](https://addons.opera.com/en/extensions/details/install-chrome-extensions/) Opera addon first
2. Visit the Chrome Web Store listing for Read It Aloud
3. Click **Add to Opera**
